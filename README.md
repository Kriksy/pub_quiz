# pub_quiz

# Grupp

Uppgift: Skapa en Interaktiv Quizapplikation

## Beskrivning:

Utveckla en interaktiv quizapplikation där användare kan testa sina kunskaper inom ett specifikt ämnesområde. Det kan vara något relaterat till teknik, popkultur, vetenskap eller vad som helst som är instressant.

## **Krav:**

1. **Hemsida Struktur:**
   - En startsida där användare kan välja ämnet för quizen. Inkludera en lista med tillgängliga ämnen och en "Starta Quiz" -knapp för varje ämne.
   - En ämnessida som förklarar mer om det valda ämnet och innehåller en ytterligare "Starta Quiz" -knapp.
   - En quizsida med ett antal frågor och flervalsalternativ.
   - Ett resultatavsnitt som visar användarens poäng och eventuellt svar på varje fråga.
2. **Interaktivitet:**
   - Använd JavaScript för att implementera själva quizen. Spara användarens poäng och visa resultatet på slutet.
   - Skapa varierande typer av frågor, inklusive textfrågor, flervalsfrågor och möjligen bildbaserade frågor.
3. **Responsiv Design:**
   - Se till att webbplatsen är responsiv och ger en bra användarupplevelse på olika enheter.
4. **CSS-styling:**
   - Använd CSS för att skapa en enhetlig och användarvänlig design för quizen.
5. **Extra Utmaning (frivillig):**
   - Implementera en tidbegränsning för varje fråga för att öka spänningen.
   - Ge feedback till användaren direkt när de svara på frågan, om det är rätt eller fel svar.

## **Bedömning:**

- Funktionalitet och interaktivitet med JavaScript
- Responsivitet
- Design och användarupplevelse
- Kodstruktur och läsbarhet
